Intelligence Guide

I. Performance Features
  • Precision Tracking: Comprehensive logging for resistance training, including load, volume, and intensity.
  • Integrated Cardio Analytics: Specialized tracking for distance, calories, and heart rate.
  • Elite Dashboard: Dynamic stats engine providing real-time personal accolades.
  • Predictive Biometrics: Integrated 1RM algorithm and Strength-to-Bodyweight ratios.
  • Smart Entry: Predictive text and Auto-Flow sets to streamline sessions.
  • Heatmap: Interactive 35-day grid visualizing training consistency.
II. Operational Procedures
  • Profile Integrity: Data is encrypted and tied to your specific email address.
  • Logging Lifting: Use "+ Add Lifting", input the first set, and use "+ Set" for auto-populated rounds.
  • Logging Cardio: Use "+ Add Cardio" for aerobic sessions. Dashboard priorities are Distance and Calories.
  • Finalization: Review all entries and select "SAVE ALL." Avoid double-tapping to prevent duplicate records.
  • Correction: Use the History panel to delete individual records or mistakes post-save.
III. Pro-Tips
  • Account Recovery: Use "Forgot Password" to receive a secure 6-digit reset code via email.
  • Cardio Formats: Record duration in decimal format (e.g., 20.5 for 20m 30s).
  • 1RM Logic: Enter any two metrics in the calculator and leave the third blank before hitting Calculate.
  • Custom Slots: Personalize the four trackers at the bottom of the Stats page via the dropdown menus.
